* else
```Python
a = 1
b = 2
if (a < b):
  print(f'{a} is less than {b}')
else:
  print(f'{a} is greater than {b}')
```
* elif
```Python
a = 1
b = 2
if (a < b):
  print(f'{a} is less than {b}')
elif (a > b):
  print(f'{a} is greater than {b}')
else:
  print(f'{a} equals {b}')
```
